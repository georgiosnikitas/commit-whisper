---------------------------------------------------------------
 Requirements
---------------------------------------------------------------
- Node.js (LTS recommended)
- npm (bundled with Node.js)
- macOS (developed and tested on macOS)

---------------------------------------------------------------
 Getting Started
---------------------------------------------------------------
1. Install dependencies:

       npm install

2. Run the project:

       npm start
